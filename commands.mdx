## Commands List

### Staff Management

`/promote`  - Issues a promotion one selected user.

`/mass promote` - Issues a promotion all users with the selected role.

`/infract` - Issues an infraction to one user (verbal warn, warn, strike or demote).

`/mass infract` - Issues an infraction all users with the selected role.

`/suspend` - Suspends a user.

### User Management

`/warn` - Issues a warning to a user.

&#x20;`/mute` - Mutes a user with the built-in discord timeout feature.

&#x20;`/kick` - Kicks a user from the server.

&#x20;`/ban` - Bans a user from the server.

&#x20;`/search` - Displays information about a user.

&#x20;`/nick` - Changes the nickname of a user.

&#x20;`/role` - Toggles a role of a user.

&#x20;`/roleall` - Toggles a role for users with the selected role.

### Utilities

`/afk` - Sets an AFK for a user.

&#x20;`/cat` - Shows an image of cats.

&#x20;`/dog` - Shows an image of dogs.

&#x20;`/giveaway` - Creates a giveaway.

### Server Management

`/config` - Configures the bot for the server.

&#x20;`/serverinfo` - Displays the information of the server.

### ER:LC Management

`/erlc info` - Displays the information of the ER:LC server.

&#x20;`/session` - Sends session messages for ER:LC server.

### Misc

`/help` - Provides help with setting up the bot.

&#x20;`/invite` - Sends the invite link to the bot.

&#x20;`/premium` - Sends the link to the premium bot.

&#x20;`/ping` - Displays the ping of the bot.

&#x20;`/about` - Displays information about the bot.

&#x20;`/vote`  - Sends the link to vote for Covert on top.gg.
